前言
前几天，偶然看到非常好看的个人博客，让我产生了浓厚的兴趣，搞了两三天，遇到了很多问题，接下来就记录一下简单的搭建过程吧，入门级别的笔记，大佬勿喷！

静态网页与动态网页简述
静态网页------无需跟服务器交流，比如纯HTML写的网页，今天搭建的博客就是这种类型
动态网页------需要跟服务器交流，比如可以登录的网站
一般网页都是两者结合使用的
托管静态网页的平台
例如github(国外),gitee(国内),coding(国内)等

静态网页生成器
hexo，hexo(便捷版)，gridea，Jekyll，Hugo等

所需工具
使用github托管个人博客静态网页，使用gridea作为静态网页生成器(操作简单)

github账号官网
gridea下载安装官网
部署个人博客
创建仓库存储静态网页

生成个人访问令牌
依次点击，右上角头像->Developer settings->Personal access tokens->Generate new token

在gridea填写域名，仓库名称等信息，同步

刷新一下github仓库网页，会看到多了很多文件

设置主题
原生主题不好看，怎么换主题呢

gridea主题市场

我挺喜欢的主题next主题地址

找到你喜欢的主题，下载即可

添加文件后，可以在gridea主题更改主题，不同主题功能不一样

配置好主题，再同步

就可以通过xxx.github.io访问博客了

绑定域名
xxx.github.io这个网址不够个性化，也不好记，于是就需要绑定域名，通过域名访问博客

注册域名
有免费的，也有收费的，这里使用阿里云注册的域名，刚开始可以买一个便宜的域名玩玩，我买了个首年1块钱的域名

阿里云域名注册地址

购买后需要实名认证，搜索域名控制台

绑定域名

点击解析

只有域名解析是访问不了博客的，需要github上部署的博客知道你是用什么域名访问

测试解析
域名解析生效测试
阿里云的dns解析网站链接

本地域名解析生效测试
由于本地DNS缓存的原因，可能需要等几分钟才能访问博客

苹果电脑下载
